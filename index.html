Intro

When starting a GitHub account, it may look too much, however if you know how it works you can easily see that it is really simple. The main use of this website is so that the students will be able to create their own websites without doing too much work. Though it may also look easy, you can easily make a mistake when you use the wrong syntax, or command. And such, I will be discussing on how you can be able to use it more efficiently.

But before all that, let's start at the beginning. Making your GitHub account. If you have already made a GitHub account, then you may move on to the next step at Info.

  1. Go to GitHub.com.
  2. Tap sign-up, and give all the necessary deatails to create your account.
  3. Remember that your username has to always be in lowercase, and no special characters with certain exceptions.
  4. Then complete your account by finishing the last few steps.
  5. You have succesfully created a GitHub account, now move on to the next page Info.

Info

After creating your GitHub account, you now have to make a repository so that you can be able to search your website in google.

How to get started:

  1. Tap the + and new repository, or tap "create new repository".
  2. Name the repository “name.github.io” (Change name to your username). (hakui27[username]/hakui27.github.io[repository name])
  3. Add a README.md file for easier editing, but if you forgot tap "add file" then "create new file". Then rename that file to "README.md" so that you can create your content.
  4. Then create repository.
      5. What to do:
      1. After you created your repository, open your repository then tap "Add File" and "Create new file".
      2. Rename the file you created to "_config.yml" so that you can add a theme to your website.
      3. After that, choose a template from GitHub Pages.
      4. In the "Edit file" section, type "theme: jekyll-theme-name of theme". (theme: jekyll-theme-merlot)
      5. Then commit change at the very bottom to finish making your theme.

Content

And now let's finish your website by making the contents.

Editing the website:
  1. Return back to your repository after you commit the “_config.yml” file.
  2. Edit the README.md file by tapping the pencil icon.
  3. Then after, start making the contents of your website. It doesnt matter in the activity.
  4. Here is a Cheat Sheet if you have trouble with making the contents like the basic syntax.
